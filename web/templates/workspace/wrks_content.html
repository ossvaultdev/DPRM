ACQUISITION


This web page will allow the user to create one or more web API requests. On this page, there will be a function to launch the web API request(s) and receive data in lists on the same page.


From this page, the user can also schedule web API requests to be automatically executed at user-defined intervals.

TRANSFORMATION


This web page will display the results from the Acquisition page in one or more data tables so that the user can transform them, modifying the data by calculating, adding, cross-referencing with other data, and reducing it to obtain the necessary dataset for their applications.

EXPLOITATION


This page allows users to specify and configure how to process the acquired and transformed data.


This includes the ability to extract files via internet links to download them in various formats, as described in the description, or to share them with other users via internet links or web API requests.


To access the Transformation and Data Processing web pages, the previous step must be completed.
